Welcome to Métamorphose-2 help!
You can keep this help dialog open and still access all other program funtions.

How to use Métamorphose-2:

  • in the Picker tab, click the BROWSE button to select a path. You may also enter in or paste a path and click the Refresh button. The contents of this path will then be displayed.

  • By default, the entire contents of the path are selected when a new path is opened, you can (de)select files or folders by clicking on them. All selections will use the same renaming rules. Any folders you select will not have their contents changed.

  • In the - Renamer - tab, select which renaming operations you want to use by double-clicking or drag-and-dropping them. You may then set the operation's parameters within the main section that will appear in the center of the window. You may also change general operation settings using the tool bar towards the top of the window, or by right-clicking an operation in the Used list.
  • Items will be previewed automatically as you change settings. If you do not like this behavior, uncheck Auto Preview, and items will only be previewed when the Preview button is clicked.

  • Errors that will prevent the renaming operation from completing succesfully are highlighted in red, warnings are in yellow. The Errors/Warnings tab will display them.

  • If preview looks OK then click the Go! button. At this point your selections will be renamed.

  • You have one undo level in case of mistakes.

Most parts of the program will display a help tip if you let your mouse cursor linger a bit over it.
Also definitly check out the other tabs of this help window to see options in more detail!
